Менеджер для расчета оптимальных отступов от краев контейнера карты.

У каждой карты есть собственный менеджер отступов. Чтобы добавить область над картой в менеджер, используйте метод addArea Метод addArea возвращает объект, представляющий занятую прямоугольную область. Он позволяет изменить, либо удалить область из менеджера объектов.

Область описывается в виде объекта, который содержит информацию о смещении от краев карты и размер. В качестве единиц измерения возможно использовать пиксели (px), проценты (%). Проценты вичисляются относительно размеров контейнера с картой.

Чтобы добавить область, занимаемую контролами в менеджер объектов, при их создании выставите опцию adjustMapMargin в значение true.
