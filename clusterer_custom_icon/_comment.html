Для объединения близко расположенных меток в группу (кластер) используется кластеризатор.

Кластеризатор реализуется классом . В качестве параметров конструктору можно передать опции кластеризатора: стиль отображения значка кластера, размер его ячейки и т. д. Список доступных опций можно посмотреть в соответствующем разделе справочника. Опции для настройки меток-кластеров задаются в кластеризатор с префиксом "cluster". Список опций меток-кластеров доступен в документации .

Вы можете задать произвольное изображение для меток кластера. Для этого нужно переопределить опцию clusterIcons.
