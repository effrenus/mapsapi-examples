В примере показано, как создать собственную карту. Карта задана в Декартовой (прямоугольной) системе координат, таким образом, координаты, указывают на положение точки на плоскости, с началом в центре (координата [0, 0]). Для географических координат есть несколько проекций: эллептическая (используется по-умолчанию) и сферическая проекция Меркатора.

После этого, необходимо подготовить исходное изображение к отображению на карте. С этой целью изображение разбивается на кусочки (тайлы), в качестве инструмента можно порекомендовать сайт http://geohosting.pro/products/tiler/

Далее создается новый слой карты, задается шаблон по которому доступны созданные тайлы, а также масштаб для которого доступны тайлы. Создается новый тип карты, для которого в списке слоев задаем ранее созданный.

Теперь мы можем создать собственную карту, для этого при вызове конструктора Map необходимо указать свой тип карты, а также проекцию в которой вы хотите отобразить карту.
