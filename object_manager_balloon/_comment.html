- класс, позволяющий добавить на карту большое количество объектов без необходимости создавать метки по отдельности.

Объекты, добавленные в ObjectManager, располагаются в коллекции ObjectManager.objects. Кластеры, образованные из добавленных меток, добавляются в коллекцию ObjectManager.clusters.

У этих коллекций есть менеджеры управления балунами и хинтами (ObjectManager.objects.balloon, ObjectManager.clusters.hint).
