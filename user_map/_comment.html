API предоставляет возможность создавать собственные карты.

Для создания собственной карты необходимо прежде всего задать систему координат, на которую будет проецироваться карта. В API встроена поддержка двух систем координат: декартовой и географической, которые реализуются классами и соответственно. Кроме того есть возможность создания пользовательской координатной системы.

Также для создания пользовательской карты нужно сформировать собственный слой и добавить его в хранилище слоев карты. Для создания слоя используется класс . При добавлении слоя в хранилище слоев указывается не экземпляр класса Layer, а его конструктор.
